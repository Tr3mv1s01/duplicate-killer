# Changelog

## [1.0.0] — 2026-07-15

### Added
- 2-phase duplicate scanning (size grouping → MD5 hash)
- Folder selection with recursive scan
- Progress bar with file/group/space stats
- Results display with per-file checkboxes
- Safe deletion to Recycle Bin (send2trash)
- Dark and Light theme support
- English and Russian interface
- Export to TXT and CSV
- Settings dialog (theme, language, exclusions)
- Excluded folders and extensions filtering
- Graceful scan cancellation
- Error handling with user-friendly dialogs
- Automatic config save in user app data
- Rotating log files (2 MB × 3 backups)
